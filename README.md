🎨 Fine Art E-Commerce Platform

A modern and scalable e-commerce platform designed specifically for artists, galleries, and art enthusiasts. The platform enables users to explore, purchase, and manage fine art collections through an intuitive and visually appealing interface.

📌 Overview

Fine Art E-Commerce Platform provides a seamless digital marketplace where artists can showcase their artwork and customers can discover unique pieces from various categories. The application focuses on user experience, secure transactions, and efficient artwork management.

🚀 Installation
Clone Repository
git clone https://github.com/your-username/fine-art-ecommerce.git
cd fine-art-ecommerce
Install Dependencies
npm install
Configure Environment Variables

Create a .env file and add the required environment variables:

MONGODB_URI=your_database_url
JWT_SECRET=your_secret_key
PORT=5000
Run Application
npm start

For development:

npm run dev
